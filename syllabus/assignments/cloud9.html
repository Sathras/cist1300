Info: Cloud9 is a browser-based IDE (development environment) that we will use for all coding activities (except the tutorials, which come with their own interactive solution).
  • Specifically we will use each student's private workspace (assignments) for individual assignment submissions
  • Specifically we will use the instructor's public workspace (cist1300) for class materials
  • Specifically we will use the instructor's public workspace (cist1300-class) for: in-class participation

Using Cloud9 as an IDE comes with several advantages:

  • Students can follow the instructor in class while looking at their own screens. This works like Google Docs and code is updated in real-time. This way students in the last rows will be able to see what the instructor is typing.
  • Cloud9 has a file history system built in and you can easily go back in time and see change after change.
  • Cloud9 has a strong color highlighting built in. This makes code much more readable.
  • There are countless other advantages you will explore and recognize during the semester.
