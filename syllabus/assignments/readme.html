Website - Readme

Submission Deadline: 01/31 10:00 AM
Maximum Score: 20
Submission via: Cloud9

Every Development Project should have a ReadMe. The purpose is to describe the project and provide important related information like contributors, external sources and more.

In this exercise the students will have to write a brief readme.md file using the mark-down language.

The readme.md file should be placed in the root of the student's private workspace and should be named README.md. You can replace the content of the sample readme.md that is always present in a new cloud9 workspace.

The description of your website topic should be sharp - that means short but very precicely capturing the content of your website.

Cheat Cheat This links shows you how you can format your markdown file and improve its visual style and quality.

A Sample Readme is now available under /downloads/assignments/README.md
For full score, please extend content and quality of your README.md file and also preview it in Cloud9.

Grading Rubric Score
Quality of ReadMe File (organized file structure, placed and named correctly, usage of at least a few markdown concepts like lists, headings, links...) 8
Quality of ReadMe Content (only significant text for description, but simulaniously complete and precise information about the intendet topic) 8
Relevance / Suiteability of topic (can it be easily accomondated in both website projects) 4
Total 20

Idea: It is probably helpful to look at the grading rubric for the three implementations of the website, to see if the intended project topic is actually suiteable for all website assignments. Keep the topic very basic and specific and you will avoid problems and extra effort later!

Sample Topics

  • Personal Website about yourself
  • Company Profile
  • Website on a specific theme (e.g. Politics, Nature,...)
  • Online Resume / CV
  • Landing Page (e.g. marketing an event)
  • Appartment for Rent
  • Information Page about a personal Hobby
  • Information Page about a personal idol (band, politician, actor,...)
