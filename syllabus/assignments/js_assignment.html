Grading Rubric Score
Clicking on the "Reset" button should put the gallery back into its original state. 10
Clicking on the "Enlarge" button should enlarge all pictures by 50% in both width and height. Clicking it again should restore the original size. Also indicate if enlargment is active by adding/removing a class "active" to the button 20
Clicking on the "Filter 1" button should hide all pictures that have their "data-filter" attribute set to 1. Clicking it again should show them again. Also indicate if the filter is on by adding/removing a class "active" at the button. 20
Bonus: Clicking on Filter 2 button should blur and grayscale all pictures that have their "data-filter" attribute set to something different than 2. Clicking it again reverses the process. Also indicate if the filter is on by adding/removing a class "active" at the button. 20
Total 50 (+20)
