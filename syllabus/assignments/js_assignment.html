The Assignment is to program a small app using jQuery. Given a html file including a Bootstrap table, add a JavaScript file, that allows you to:

  • sort tabled by each column (asc and dsc)
  • hide/show individual columns
  • reset the table to the default view

You wont have to alter the existing html file to enable all the functionality described above.

Advice: You might first want to get the data already present in the table and add it to a JavaScript object that can then be sorted and filtered.
