Overview In this course, we will learn how to create static web pages using HTML5, CSS and JavaScript. We will also learn how to make use of a popular styling library (Bootstrap), in order to create visually professional looking, mobile-friendly websites. We will further cover a powerful JavaScript library (jQuery) to make web applications and games like chess, snake and more.
Prerequisite MATH 1310 (or equivalent) is required. Previous experience in programming, data structures and/or fast keyboard typing will come in handy.
